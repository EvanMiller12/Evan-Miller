Delta Machinery Parts

Delta Machinery Parts is a web platform that allows consumers to create an account, search, review specifications/documentation, and purchase a part or parts for their machine. The Node API interfaces with an ERP system database containing all order and machine to part relationships, keeping information readily available and consistent for the ERP system users.

  • Webpack
  • babel
  • Authentication
  • e-commerce
  • Google analytics & maps
  • Node
  • Authorization
  • Cloud Hosting
  • CI/CD

DegreeScale

DegreeScale is a web app built designed to help new and current students choose the right college major and degree, the first time. It provides salary statistics from the College Scorecard database to compare salaries between an Associates and Bachelors degree of the same major. DegreeScale also allows creation of user profiles and reviews. A proxy server is was setup to get around a CORS issue with the College Scorecard API.

Technologies

  • React
  • React-Bootstrap
  • Backbone
  • jQuery
  • Node
  • CORS
  • Parse Server
  • Heroku
  • CI/CD

The Simpsons TBG

The Simpsons TBG is a turn based game built for a project built for school to learn constructors and prototype's. Users can pick their character from a dropdown list and choose different strategies of attack. The attacks apply different levels of damage to their opponent, but the more potential damage an attack has corresponds to a less chance of actually hitting/damaging the opponent.
